API Documentation

Authentication

All API requests require your token in the Authorization header:

Authorization: your-token-here

Face Comparison API

POST /compare

Request Format (multipart/form-data)

Parameter Type Required Description
reference_image File Yes Reference face image (jpg, png)
live_image File Yes Live face image to compare
user_id String Yes Your unique identifier
retention_period Integer Yes Days to retain data (e.g., 30)

Code Examples

cURL
curl -X POST http://your-server/compare \
-H "Authorization: your-token-here" \
-F "reference_image=@ref.jpg" \
-F "live_image=@live.jpg" \
-F "user_id=your-user-id" \
-F "retention_period=30"
Python
import requests

url = 'http://your-server/compare'
headers = {'Authorization': 'your-token-here'}

files = {
    'reference_image': open('ref.jpg', 'rb'),
    'live_image': open('live.jpg', 'rb')
}

data = {
    'user_id': 'your-user-id',
    'retention_period': 30
}

response = requests.post(url, 
    headers=headers,
    files=files, 
    data=data
)
JavaScript
const formData = new FormData();
formData.append('reference_image', refImageFile);
formData.append('live_image', liveImageFile);
formData.append('user_id', 'your-user-id');
formData.append('retention_period', '30');

fetch('http://your-server/compare', {
    method: 'POST',
    headers: {
        'Authorization': 'your-token-here'
    },
    body: formData
})
.then(response => response.json())
.then(data => console.log(data));
